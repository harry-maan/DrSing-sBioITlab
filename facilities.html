Facilities at BioIT Lab

The Dr Singh's BioIT lab offers various facilities, Including Biosafety Lavel 2 Facility, Data Science Facility, Computational Structural Biology, and NGS Data Analysis

Biosafety Lavel 2 Facility

Antibiotic Susceptibility Test (Ast): BioIT lab has established AST facility for culturable bacterial strains of BSL2 level organisms.

Data Science Facility

Advanced Statistical Analysis: Dr Singh's BioIT has computational facility to deliver advanced statistical analysis on demand.

We covers:

Sample Size Calculations

Hypothesis Testing

AUC/ROC Analysis

Correlation and Regression Analysis

Meta-analysis and Network Meta-analysis

Computational Structural Biology

Computer Aided Drug Discovery: Dr Singh's BioIT has advance GPU-accelerated workstation dedicated for CADD and Molecular modeling.

We covers:

Virtual Screenings (SBVS and LBVS)

Molecular Dynamics Simulations

Meta-dynamics Simulation

Umbrella Sampling

Trajectory Analysis

Free energy calculations (MM/PBSA and FEL)

NGS Data Analysis

Genomics Data Analysis: Dr Singh's BioIT has dedicated workstation for NGS data analysis.

We covers:

Single Neucleotide Polymorphism (SNP)/SNV Analysis

RNA-Seq data analysis

Mutation Annotated File plotting

Gene Ontology/Pathway Enrichment Analysis
